# Gestor de Tareas
Aplicación web sencilla para la gestión de tareas, donde se puedan: Crear, editar, eliminar y listar tareas. Asignar un responsable activo. Registrar la fecha de asignación, y validar que solo los usuarios activos puedan ser responsables.

## Su funcionalidad consiste en

**Organización:** Permite crear listas de tareas y la descripcion del mismo

**Asignación de responsabilidades:** Facilita la asignación de tareas a responsables específicos del equipo .

**Seguimiento del progreso:** Muestra el estado de cada tarea.

**Planificación:** Incluye la capacidad de establecer plazos, fechas de entrega.

**Priorización:** Ayuda a establecer prioridades para enfocarse en las tareas más importantes.

## Conclusiones
Un gestor de tareas es una solución tecnológica diseñada para optimizar el seguimiento y la administración de las actividades operativas dentro de una organización. Su objetivo es fortalecer el control interno y mejorar la eficiencia operativa a través de un manejo centralizado y ordenado de las tareas corporativas.
